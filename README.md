# Resale App

A marketplace app for buying and selling secondhand items with ease.

## Features

- Browse listings by category, price, and condition
- Snap photos and post an item for sale in minutes
- In-app messaging between buyers and sellers
- Saved searches and favorites
- Secure checkout and order tracking

## Getting Started

1. Clone the repo
2. Install dependencies
3. Configure your environment variables
4. Run the app locally
